About

Web3 Pi offers simple, step-by-step instructions to build your own Ethereum node using various Raspberry Pi products. Set up your node on these low-power, small form factor devices for faster and easier access to the Ethereum blockchain.

Supported models:

  • Raspberry Pi 5
  • Raspberry Pi 4
  • Raspberry Pi Compute Module 4
